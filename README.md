- TailwindCSS

## Quick Start

1. Clone this repo and install the dependencies

```shell
npm i
```

2. Run the project

```shell
  npm run develop
```

### Cypress for testing

I have created a single task for cypress which is located in `/cypress/integration/examples/init.spec.js`. This is just to test the UI and pages.
